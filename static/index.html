Controls

  • A, 4, ← = Left
  • S, 2, ↓ = Down
  • D, 6, → = Right
  • W, 8, ↑ = Up
  • SPACE, Right Click = Fire
  • F5 = Reset Game
  • F11 = Full Screen

Info

  • Score points by destroying enemy ships
  • Player does not lose health when flashing
  • If the Bloodcell Count goes below zero, the game is over
  • The timer doesn't do anything yet
